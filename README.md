# BrainBoost - Java Learning App 🚀

BrainBoost is an AI-powered Java learning application designed to help beginners learn Java in an interactive and smart way.

## ✨ Features
- Daily Java Concepts
- Java Quiz
- Quick Revision Notes
- AI Java Notes
- Interactive GUI

## 🛠 Tech Stack
- Java
- Swing GUI
- OOPs
